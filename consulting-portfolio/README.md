# Consulting Portfolio (anonimizado)
Este repositorio agrupa **casos de estudio** y walkthroughs (sin código) para demostrar experiencia en diseño, backend, datos, IA y optimización.
## Índice de proyectos
- **P01** — Framework modular de forecasting y señales (series temporales) (`projects/P01_forecasting_signals_framework`)
- **P02** — Plataforma unificada para bots + análisis (orquestación de módulos) (`projects/P02_unified_trading_ai_platform`)
- **P03** — Backend seguro para servicios transaccionales (API) (`projects/P03_secure_trading_backend_api`)
- **P04** — Demo web (Vite+TS) para SaaS de gimnasio con ingreso QR (local) (`projects/P04_gym_saas_qr_demo`)
- **P05** — Landing minimalista para venta de productos digitales (funnel) (`projects/P05_digital_products_landing_funnel`)
- **P06** — Optimización de secuenciación productiva expuesta vía API (`projects/P06_production_sequencing_optimizer_api`)
- **P07** — Plataforma de envío de correos (campañas) + deliverability (`projects/P07_email_delivery_platform_bulk_campaigns`)
- **P08** — API LLM+RAG para análisis de comunicaciones (NLP) (`projects/P08_llm_rag_comms_intelligence_api`)
- **P09** — Workflows RAG para documentos legales (clasificación y etapas) (`projects/P09_legislative_rag_admissibility_workflows`)
- **P10** — Orquestador tipo MCP para agentes + herramientas (n8n/Telegram) (`projects/P10_mcp_orchestrator_telegram_n8n`)
- **P11** — Análisis de discurso público: sentimiento/toxicidad + RAG (`projects/P11_social_discourse_sentiment_toxicity_rag`)
- **P12** — Recolección multi‑plataforma + persistencia en PostgreSQL (`projects/P12_social_media_scraping_postgres_pipeline`)
- **P13** — Scraper/ETL de documentos públicos + API de consulta (`projects/P13_public_institution_scraper_pipeline`)
- **P14** — Harvester de noticias + clustering/temas + generación de reporte (`projects/P14_news_harvester_topic_pipeline`)
- **P15** — ML para análisis de sensibilidad + surrogate models (resultados de optimización) (`projects/P15_industrial_ml_sensitivity_surrogate`)
- **P16** — Optimización multi‑etapa: relajación, fijación de binarios e infeasibilidad (`projects/P16_multistage_optimization_binary_fixing`)
- **P17** — Editor interactivo de redes (ReactFlow) con nodos/handles custom (`projects/P17_interactive_network_editor_custom_nodes`)

## Cómo leer
- Parte por `projects/PXX.../README.md`
- Luego `ARCHITECTURE.md` y `RISK_AND_SECURITY.md`
- Finalmente el notebook `notebook/WALKTHROUGH.ipynb`
