# P03 — Backend seguro para servicios transaccionales (API)

## Resumen
API con autenticación, roles y endpoints transaccionales, con foco en seguridad, validación y límites operativos.

**Stack:** Python (API), auth con tokens, DB relacional, contenedores.

## Qué se entregó
- Documentación técnica (arquitectura, decisiones, trade‑offs)
- Walkthrough (notebook sin código)
- Recomendaciones de operación y seguridad

## Señales de seniority
- Modelo de permisos (admin vs usuario)
- Validación de inputs y manejo de errores consistente
- Estructura preparada para CI/CD

## Límites por confidencialidad
- No se incluye implementación, datos reales, ni endpoints
- Nombres propios y dominios fueron reemplazados por placeholders
