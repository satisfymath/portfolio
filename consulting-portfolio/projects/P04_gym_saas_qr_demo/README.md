# P04 — Demo web (Vite+TS) para SaaS de gimnasio con ingreso QR (local)

## Resumen
Mockup interactivo para mostrar flujo de ingreso por QR, panel admin y control de pagos, todo en modo local para demo comercial.

**Stack:** Vite, React, TypeScript, LocalStorage, QR scanning (web), GitHub Pages.

## Qué se entregó
- Documentación técnica (arquitectura, decisiones, trade‑offs)
- Walkthrough (notebook sin código)
- Recomendaciones de operación y seguridad

## Señales de seniority
- UX minimalista y responsive
- Separación de rutas (admin/user)
- Persistencia local para demo sin backend

## Límites por confidencialidad
- No se incluye implementación, datos reales, ni endpoints
- Nombres propios y dominios fueron reemplazados por placeholders
